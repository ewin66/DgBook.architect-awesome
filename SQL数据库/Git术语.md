# git 拉取(pull)和获取(fetch)的区别



**git  pull**   从远程拉取最新版本 到本地  自动合并 merge       git pull origin master

**git  fetch**  从远程获取最新版本 到本地  不会自动合并 merge   git fetch  origin master   

实际使用中  使用**git fetch 更安全**  在merge之前可以看清楚 更新情况  再决定是否合并
